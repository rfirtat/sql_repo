# CREATE
`CREATE DATABASE DatabaseName` creates a new database  

```SQL
CREATE TABLE [IF NOT EXISTS] TableName  (
    Column1 DATA_TYPE,
    Column2 DATA_TYPE,
    ...
    ColumnN DATA_TYPE
);
```




# DELETE
`DROP DATABASE DatabaseName` deletes a **database**, including all tables  
`DROP TABLE [IF EXISTS] TableName` deletes a **table**, along with all the table's rows
