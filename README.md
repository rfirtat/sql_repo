# SQL Notes and Practice

## Overview

This repository contains my **notes, practice problems, and small projects related to SQL and relational databases**. The goal of this repository is to build a strong foundation in querying, database design, and data analysis using SQL.

The material in this repository is a combination of:

- Personal notes on SQL concepts
- Practice queries and exercises
- Database design examples
- Small projects used to apply SQL concepts

---

## Topics Covered

Some of the core topics explored in this repository include:

- Basic SQL queries (`SELECT`, `WHERE`, `ORDER BY`)
- Filtering and conditional logic
- Aggregate functions (`COUNT`, `SUM`, `AVG`)
- Grouping data (`GROUP BY`, `HAVING`)
- Joins
- Subqueries
- Database schema design
- Relational database concepts

---

```
notes/
```
